Workshop materials

Practice II: Building Scientific Networks from Web of Science

Summary:
Participants use Web of Science records to construct two network types: coauthorship networks (authors connected through joint publications) and coreference networks (papers connected through shared cited references). 

Practice III: Measuring Disciplinary Hardness and Softness

Summary:
Participants analyze the networks created in Practice II using structural and epistemic indicators, including average degree centrality, average betweenness centrality, and average clustering coefficient measures.
